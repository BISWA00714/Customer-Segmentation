# Customer Segmentation Using Machine Learning

This project focuses on Customer Segmentation using Machine Learning techniques to analyze customer behavior and divide customers into meaningful groups based on their purchasing patterns, annual income, spending score, and other attributes.

The main objective of this project is to help businesses understand different customer categories so they can create targeted marketing strategies, improve customer satisfaction, and increase business growth.

The project uses clustering algorithms such as K-Means Clustering to identify similar customer groups from the dataset. Data visualization techniques are also used to better understand customer distribution and cluster patterns.

## Features
- Data preprocessing and cleaning
- Exploratory Data Analysis (EDA)
- Customer behavior analysis
- K-Means Clustering implementation
- Optimal cluster selection using Elbow Method
- Data visualization using charts and graphs
- Customer group prediction
